A11y Learning: Making the Web Accessible to Everyone

Challenge

Many designers and developers do not apply accessibility guidelines due to the complexity and lack of time to study the subject.

Details

Role: UX Researcher, UX Designer, Visual Designer, Accessibility Specialist

Team: Solo Designer

Duration: 3 months

My Responsibilities

As the solo designer on the A11y Learning project, I led the end-to-end design process. I conducted user research, including surveys and competitive analysis, to identify key barriers to accessibility. I created personas, user stories, and user flows to inform the design strategy. I developed sketches, wireframes, high-fidelity mockups, and iterated on prototypes based on usability testing feedback. Additionally, I crafted the branding elements, including the logo, color palette, and typography, to ensure a cohesive user experience.

User Stories

I created and prioritized user stories to guide the development of the app. These priorities ensure that the most critical features are implemented first to provide the best user experience.

  • As a user, I want to sign up for an account so I can use the app.
  • As a user, I want to sign in to my account so I can access my personalized settings and progress.
  • As a user, I want to start a quiz so I can learn about accessibility.
  • As a user, I want to read links about accessibility so I can stay informed.
  • As a user, I want to create notes to remember what I have learned and check them at any time.
  • As a user, I want to update my learning goals to maintain consistency in my learning process.
  • As a user, I want to see my achievements so I stay motivated to continue learning.

  • As a user, I want to read a tutorial so I understand how the app works.
  • As a user, I want to choose the quiz level so I can continue growing in my studies.
  • As a user, I want to set up my password so I can secure my information.

  • As a user, I want to update my personal information so I can receive notifications.
  • As a user, I want to turn notifications on or off to control my learning reminders.
  • As a user, I want to receive notifications to remind me to learn and keep up the good work.

Branding and Identity

Name

I aimed to include "accessibility" and explain the app's purpose in the name. "Accessibility Learning" felt too long, so I chose "A11y," a shorthand for "Accessibility" (A + 11 letters + Y), emphasizing simplicity and inclusivity.

Color Palette

For the primary color, I chose blue, representing calm, relaxation, stability, trust, and loyalty. The secondary color, pink, symbolizes love, kindness, and compassion.
